Front End Developer Test

Для понимания Вашего уровня и стиля, просьба выполнить следующее тестовое задание.

Задача:

  • Сделать поиск по игрокам, используя дизайн (перевести дизайн в верстку HTML и CSS)

Минимальные требования:

  • В реализации просьба использовать HTML, CSS и JavaScript.
  • Дизайн реализовать исходя из предложенного макета PSD.
  • Используйте AJAX.
  • Логика поиска должна быть реализована на JavaScript.
  • Поиск по запросу Papy Djilobodji должен вывести игрока "Papy Djilobodji"
  • Ваше решение будет проверено на сервере с установленными Ubuntu, Nginx и PHP 5.5
  • Решение просьба присылать в zip архиве или ссылкой на github
  • Ваше решение будет залито на сервер и будет проверено по ссылке http://testserver/yourname/

Будет плюсом:

  • Написание кода с возможностью реиспользования.
  • Точность поиска.
  • Сортировка результатов по релевантности.
  • Поддержка точных совпадений, например запрос Central Midfield "England", должен вывести "Ruben Loftus-Cheek", но не "Cesc Fabregas" или "Ramires".
  • Поиск по нескольким полям, например запрос Brazil Midfield должен вывести всех игроков из Бразилии на позиции "Midfield".
  • Поддержка исключений, например запрос Brazil -Midfield, должен вернуть "Willian" и "Kenedy", но не "Ramires" или "Oscar".

Дополнительно:

  • Проверять свое решение. Если оно не заработает после запуска на сервере - мы не будем танцевать с бубном, чтобы его запустить.
  • Можно использовать Framework-и. Если требуется дополнительная настройка перед выполнением, просьба написать инструкцию.
  • Комментарии к коду очень приветствуются.
